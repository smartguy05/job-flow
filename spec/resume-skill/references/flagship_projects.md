# Anthony's Flagship Open Source Projects

Always include **at least one** of these in every resume. Include others if relevant to the target role.

## Net-Guardian-AI (Python/TypeScript)
AI-powered home network security monitoring system. Features:
- Multi-source log collection (API polling, file watching, HTTP push, UDP NetFlow/sFlow)
- Device inventory with auto-discovery and network topology visualization
- Behavioral baseline anomaly detection
- Claude-powered threat analysis
- Active response: AdGuard Home and router-level device quarantine
- Ollama LLM monitoring for detecting prompt injection, jailbreaks, and LLM-malware
- Threat intelligence feed integration
- 2FA/TOTP authentication
- Real-time WebSocket updates
- Prometheus metrics
- Automated CI/CD pipelines

**Best for**: Security roles, AI/ML roles, DevOps roles, anything mentioning threat detection or network monitoring

## ai.orchestrator (C#)
AI agent orchestration framework with plugin architecture enabling AI systems to interact with personal services and external tools.

**Best for**: AI engineering roles, platform/infrastructure roles, anything involving agent systems or tool use

## DataQuery Pro (TypeScript)
AI-powered CLI tool enabling natural language querying of PostgreSQL, MySQL, SQL Server, and SQLite databases with:
- Schema introspection
- AI-generated descriptions
- Self-correcting queries

**Best for**: Developer tools roles, CLI tool experience, database-related roles, AI-assisted development

## support_channel (TypeScript)
RAG-based support channel system enabling easy creation of knowledge bases with custom data uploads for contextual AI responses.

**Best for**: AI engineering, RAG experience, customer-facing AI tools

## Kagi-FastGPT (C#)
Command-line interface tool for accessing Kagi's FastGPT AI service.

**Best for**: CLI tools experience (less impressive than the others, use sparingly)
